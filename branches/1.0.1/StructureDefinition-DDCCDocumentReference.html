Resource Profile: DDCC Document Reference

Official URL: http://smart.who.int/ddcc/StructureDefinition/DDCCDocumentReference Version: 1.0.1
Active as of 2024-12-17 Computable Name: DDCCDocumentReference

A DDCC Document Reference is a document reference to a DDCC Document containing the DDCC. The content attachment url will point to a DDCC Document.

Usage:

  • Use this Resource Profile: DDCC Provide Document Bundle and DDCC:VS Add Bundle
  • Refer to this Resource Profile: DDCC Folder and DDCC Submission Set
