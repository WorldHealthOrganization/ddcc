Resource Profile: Patient (IPS)

Defining URL:http://hl7.org/fhir/uv/ips/StructureDefinition/Patient-pre-uv-ips
Version:1.0.0
Name:PatientUvIps
Title:Patient (IPS)
Status:Active as of 2021-10-04 02:09:43+0000
Definition:

This profile represents the constraints applied to the Patient resource by the International Patient Summary (IPS) FHIR Implementation Guide and describes the minimum expectations for the Patient resource when used in the IPS composition or in one of the referred resources.

Publisher:WHO
Source Resource:XML / JSON / Turtle

The official URL for this profile is: 

http://hl7.org/fhir/uv/ips/StructureDefinition/Patient-pre-uv-ips
This profile has been temporary added to this Implementation Guide, to allow the pre-adoption of some changes not yet officially published.
This profile is not an official IPS profile and it will be removed as soon as the next HL7 FHIR IPS release will be published.
