Resource Profile: DDCC Questionniare

Defining URL:http://worldhealthorganization.github.io/ddcc/StructureDefinition/DDCCQuestionnaire
Version:0.3.1
Name:DDCCQuestionnaire
Title:DDCC Questionniare
Status:Active as of 1/25/22, 2:31 PM
Definition:

Describes the logical structure for a questionnaire associacted to a DDCC which includes a Structure Map to generate the relevant DDCC from a given DDCC QuestionnaireResponse which responds to the DDCC Questionnaire.

Publisher:WHO
Source Resource:XML / JSON / Turtle

The official URL for this profile is: 

http://worldhealthorganization.github.io/ddcc/StructureDefinition/DDCCQuestionnaire
