Resource Profile: DDCC Questionnaire Response ( Abstract )

Official URL: http://smart.who.int/ddcc/StructureDefinition/DDCCQuestionnaireResponse Version: 1.0.0
Draft as of 2023-11-06 Computable Name: DDCCQuestionnaireResponse

Describes the logical structure for a questionnaire response associacted to a DDCC Questionaire.

Usage:

  • Use this Resource Profile: DDCC Generate Health Certificate Parameters, DDCC Submit Health Event Request and DDCC:VS Add Bundle
  • Examples for this Resource Profile: QuestionnaireResponse/DDCC-VS-QuestionnaireResponse-Arabic, QuestionnaireResponse/DDCC-VS-QuestionnaireResponse-Chinese, QuestionnaireResponse/DDCC-VS-QuestionnaireResponse-Example, QuestionnaireResponse/DDCC-VS-QuestionnaireResponse-French... Show 2 more, QuestionnaireResponse/DDCC-VS-QuestionnaireResponse-Russian and QuestionnaireResponse/DDCC-VS-QuestionnaireResponse-Spanish
