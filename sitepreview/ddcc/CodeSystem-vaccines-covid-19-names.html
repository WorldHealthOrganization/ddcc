CodeSystem: Covid-19 Vaccine Codes for EU DCC

Official URL: https://ec.europa.eu/health/documents/community-register/html/ Version: 1.0.0
Active as of 2022-09-13 Responsible: WHO Realm: Computable Name: EU_DCC_CodeSystem

EU DCC codes for Covid-19 vaccines so the FHIR server can perform expansions.

This Code system is referenced in the content logical definition of the following value sets:
