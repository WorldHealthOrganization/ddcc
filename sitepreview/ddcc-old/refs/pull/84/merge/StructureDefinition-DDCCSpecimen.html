Resource Profile: DDCC Specimen

Defining URL:http://worldhealthorganization.github.io/ddcc/StructureDefinition/DDCCSpecimen
Version:1.0.0
Name:DDCCSpecimen
Title:DDCC Specimen
Status:Active as of 2022-11-23 04:26:34+0000
Definition:

A DDCC Specimen contains the content corresponding to a row in the Test Events section of a Paper DDCC. While this is not a further profile of the International Patient Summary (IPS) Specimen resouce, this is intended to map into an IPS Specimen resource.

Publisher:WHO
Source Resource:XML / JSON / Turtle

The official URL for this profile is: 

http://worldhealthorganization.github.io/ddcc/StructureDefinition/DDCCSpecimen
