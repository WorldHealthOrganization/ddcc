Resource Profile: DDCC Immunization Recommendation

Official URL: http://smart.who.int/ddcc/StructureDefinition/DDCCImmunizationRecommendation Version: 1.0.0
Draft as of 2023-11-06 Computable Name: DDCCImmunizationRecommendation

A DDCC Immunization Recommendation contains the content corresponding to when a patient should return for a booster shot.

Usage:

  • Use this Resource Profile: DDCC:VS Add Bundle and DDCC:VS Document
  • Refer to this Resource Profile: DDCC:VS Composition
  • Examples for this Resource Profile: ImmunizationRecommendation/DDCC-ImmunizationRecommendation-Arabic, ImmunizationRecommendation/DDCC-ImmunizationRecommendation-Chinese, ImmunizationRecommendation/DDCC-ImmunizationRecommendation-English, ImmunizationRecommendation/DDCC-ImmunizationRecommendation-French...
